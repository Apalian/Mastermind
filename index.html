    Le Mastermind

    Le Mastermind qu'est ce que c'est ?

    Quel est le principe du jeu ?

    Le but est de trouver un code de 4 couleurs alétoires définit aléatoirement. Pour se faire le joueur doit choisir lui aussi un code de 4 couleurs afin de déterminer quels couleurs sont bien placées ou juste de la bonne couleur.

    Comment jouer ?

    Durant le premier tour, le joueur n'a aucune information sur le code a trouver. Pour se faire, il va devoir faire au hasard. Ensuite, à la fin du tour, si une des couleur est pareil au code secret alors il va recevoir une information lui disant qu'il a trouver une ou plusieurs couleurs. Si la couleur est juste et bien placé alors, il recevra une information lui disant qu'il a trouver une couleur bien placé.
